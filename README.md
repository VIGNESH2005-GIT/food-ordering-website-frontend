# 🍔 Food Ordering Website - Frontend

A modern, responsive frontend application for a food delivery platform. This project showcases interactive UI components, modular JavaScript logic, and a seamless multi-page user flow.

## 🚀 Key Features
* **Interactive Menu:** Dynamic food item selection.
* **Cart Logic:** Dedicated cart management functionality.
* **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, Desktop).
* **Clean Architecture:** Separated concerns with modular CSS and JS files.

## 🛠️ Tech Stack
* **Core:** HTML5, TailWindCSS, JavaScript (ES6)
* **Environment:** Node.js & NPM
